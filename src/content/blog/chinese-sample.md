这是一篇用于验收 PRD 的示例：正文为中文，包含**行内公式**、**块级公式**与 **代码高亮**。

质能方程（行内）：$E = mc^2$。

定积分示例（块级）：

$$
\int_0^1 x^2 \, dx = \frac{1}{3}
$$

勾股定理：$a^2 + b^2 = c^2$。

下面是 JavaScript 代码块（带语言标注，构建期由 Shiki 高亮）：

```js
function greet(name) {
	console.log(`你好，${name}`);
}

greet('世界');
```

如需在正文里写美元符号，注意与 `$...$` 数学语法的冲突，可使用转义或改用 `\(...\)` / `\[...\]`（取决于 remark-math 配置）。
